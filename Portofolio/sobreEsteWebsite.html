Tecnologias Utilizadas

HTML

Acrónimo: HyperText Markup Language

Ano de criação: 1993

Criador: Tim Berners-Lee

Site Oficial

HTML é a linguagem padrão para a criação de páginas web. É usada para definir a estrutura e o conteúdo de uma página, usando elementos como cabeçalhos, parágrafos, imagens, links, entre outros.

CSS

Acrónimo: Cascade Style Sheet

Ano de criação: 1996

Criador: Håkon Wium Lie, Bert Bos

Site Oficial

CSS é uma linguagem de estilos utilizada para definir a aparência das páginas web. Permite definir propriedades como cores, fontes, tamanhos e posições de elementos HTML.

Python

Ano de criação: 1991

Criador: Guido van Rossum

Site Oficial

Python é uma linguagem de programação de alto nível, orientada a objetos e de propósito geral. É usada para desenvolvimento web, científico, de jogos, entre outros.
