Tecnologias Web

Back-end

Laravel

  • Acrónimo: -
  • Ano de criação: 2011
  • Criador: Taylor Otwell
  • Site oficial

Laravel é um framework PHP livre e open-source criado por Taylor Otwell para o desenvolvimento de sistemas web que utilizem o padrão MVC (Model-View-Controller).

ASP.NET

  • Acrónimo: -
  • Ano de criação: 2002
  • Criador: Microsoft
  • Site oficial

ASP.NET é um framework da Microsoft para o desenvolvimento de aplicações web, que permite a criação de páginas dinâmicas, serviços web e aplicações web.

Spring MVC

  • Acrónimo: -
  • Ano de criação: 2002
  • Criador: Pivotal Software
  • Site oficial

Spring MVC é um framework Java para o desenvolvimento de aplicações web que segue o padrão de arquitetura MVC (Model-View-Controller) e possui uma grande integração com o ecossistema Spring.

Express

  • Acrónimo: -
  • Ano de criação: 2010
  • Criador: TJ Holowaychuk
  • Site oficial

Express é um framework para aplicativos web Node.js que fornece recursos avançados para lidar com solicitações HTTP e rotas.

Django

  • Acrónimo: -
  • Ano de criação: 2005
  • Criador: Django Software Foundation
  • Site oficial

Django é um framework web em Python que permite a criação rápida de aplicações web seguras e escaláveis.
